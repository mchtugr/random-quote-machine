## Run locally

Clone the project

```bash
  git clone https://github.com/mchtugr/random-quote-machine.git
```

Go to project directory

```bash
  cd random-quote-machine
```

Install dependencies

```bash
  npm install
```

Run the app

```bash
  npm run start
```
